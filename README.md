## Run app

### Don't forget environment variable

### Commands

-   Install dependencies

```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    # or
    bun install
```

-   Run app

```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    # or
    bun dev
```
